# 使用手册

最短使用方式：

1. 先读相关 wiki
2. 用 `/design` 把不稳定的设计收敛到一个 plan
3. 用 `/execute` 按 plan 推进实现
4. 用 `/bug` 处理游玩中发现的窄范围问题

维护时常用：

```powershell
pwsh -File <path-to-designvault-wiki-maintain>\scripts\wiki_lint.ps1 -VaultRoot <vault-root>
pwsh -File <path-to-designvault-wiki-maintain>\scripts\wiki_search.ps1 -VaultRoot <vault-root> -Query "keyword"
pwsh -File <path-to-designvault-wiki-maintain>\scripts\wiki_suggest_writeback.ps1 -VaultRoot <vault-root> -SourcePath ".\10 Studio\Execution Plans\样例 - 示例功能计划.md"
```
